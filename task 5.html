To ensure the exact rendering of your text on most of the browser or operating systems you have to define your fonts very carefully. The font-family CSS property can hold several font names as a fallback system. Start with the font that you want first, then the fonts you might want to fill in for the first if it is not available.You should always end the list with a generic font family, which are five: serif, sans-serif,monospace, cursive and fantasy. The generic font family allows the browser to select a similar font, if no any fonts defined by your are available.
